# mastra-agent example

Shows how to wrap `TurboMemory` as a Mastra memory provider via `@turbomem/mastra`.

```bash
export OPENAI_API_KEY=sk-...
pnpm --filter @turbomem/example-mastra-agent start
```

> Check your installed `@mastra/core` version's `MastraMemoryProvider` interface — the
> structural shape exported here is intentionally minimal.
